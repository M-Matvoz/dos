# DOS Matlab repo

Repo za vaje la DOS

## Requirements

- MATLAB R20XXa or newer

## Initialization

Clone the repository and navigate to the project folder:

```bash
git clone https://github.com/M-Matvoz/dos.git
cd dos
```

## Usage

```bash
cd $folder
```

```matlab
call_functions_here
```
## Example

```matlab
cd vaja1
fnGenerateSinus(20)
```
